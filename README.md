# Project Description
This project is an end-to-end data analytics solution built using Microsoft Excel, focusing on analyzing hotel booking data to identify cancellation patterns, customer behavior, and revenue insights.

## Problem Statement

Hotels face significant challenges due to:

High booking cancellations

Unpredictable customer behavior

This project aims to analyze these issues and provide actionable insights.

# Key Insights

High cancellation rate (~30–35%) impacts revenue

City hotels generate more bookings but also more cancellations

Couples are the largest customer segment but also cancel the most

Families are more stable and contribute to longer stays

Peak seasons (May–August) show high bookings and cancellations

Room preference does not significantly reduce cancellations

# Business Recommendations

Introduce stricter cancellation policies

Focus on family segment for stable revenue

Optimize pricing during peak seasons

Apply booking controls for high-risk customers

Encourage non-cancellable or prepaid bookings

# Conclusion

This project demonstrates how Excel can be used to transform raw data into meaningful insights. The analysis highlights that increasing bookings alone is not sufficient, as high cancellation rates directly impact business performance.
